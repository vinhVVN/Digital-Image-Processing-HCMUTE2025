Để có thể chạy chương trình ta phải cài đầy đủ các thư viện trong file requirements.txt
English: To run the program, you must install all the libraries listed in the requirements.txt file.


use this: (open cmd)
pip install -r requirements.txt

Sau đó chạy lệnh sau để khởi động ứng dụng:
Then run the following command to start the application:
streamlit run MainHome.py
